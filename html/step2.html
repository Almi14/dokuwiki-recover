Reset DokuWiki

So your environment looks good, but your wiki is still broken? Let's see what we can do.

First of all: if you still can log into your wiki and use the admin interface, it's not broken in a way that requires this script. Delete this script and fix stuff manually. Seriously, using this script will create more work for you than just fixing manually. You have been warned.

Now, if you continue, this is what will happen:

  1. switch back to the default template
  2. switch the language to English
  3. disable all plugins except the bare minimum
  4. ensure ACLs are enabled
  5. switch to the authplain Authentication backend
  6. turn the group dokuwiki-recovery into superusers
  7. create a new user in that group (we'll show you the user and password)

This should turn your wiki back to the most barebone functionality again. Enough to log in with the new user and fix things manually.
