DokuWiki Problems, eh?

This script helps you to reconfigure your DokuWiki to a state that should allow you to login again and manually fix things.

Important: Having this script on your server is a security risk. Be sure to remove it as soon as you're done with it.

This script has two steps:

  1. Check Requirements
  2. Reset DokuWiki

None of the steps will touch your actual data (pages, media) so you don't need to worry.

Using the script happens on your own risk. Please keep in mind that this script will not work for you when…

  • …certain files are not writable (the script will tell you about it)
  • …you moved your conf directory to a different place (the script might think it succeeded, but it will still not work for you)
  • …your PHP doesn't work at all (you will see some message further up)
  • …the requirements to run DokuWiki aren't met (Step 1 will help you with that)

The next step will simply check if your PHP setup matches all the requirments of the newest DokuWiki release. Maybe your setup is simply missing a little something? That step will not modify anything, it just checks things so its perfectly save to click that button :-).
