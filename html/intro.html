This script helps you to reconfigure your DokuWiki to a state that should allow you to login again and manually fix things.

This script has two steps:

  1. Check Requirements
  2. Reset DokuWiki

None of the steps will touch your actual data (pages, media) so you don't need to worry.

Using the script happens on your risk. Please keep in mind that this script will not work for you…

  • …when certain files are not writable (the script will tell you about it)
  • …when you moved your conf directory to a different place (the script might think it succeeded, but it will still not work for you)
  • …when your PHP doesn't work at all (you will see some message further up)
  • …when the requirements to run DokuWiki aren't met (Step 1 will help you with that)
